# 第一章 随机事件和概率
## 随机事件与样本空间
### 试验
在概率论中，试验是指在一定条件下进行的一次观察或测量过程，其结果具有**不确定性**。
### 样本空间
样本空间是指所有可能结果的集合，通常用符号 $\Omega$ 表示。
### 样本点
样本点是样本空间中的每一个具体结果，用符号 $\omega$ 表示。
### 随机事件
随机事件是样本空间的子集，表示试验结果满足某种特定条件的情况。
比如，掷一枚骰子，样本空间为 $\Omega = \{1, 2, 3, 4, 5, 6\}$，事件“掷出偶数”可以表示为 $A = \{2, 4, 6\}$。

## 事件的关系和运算
### 事件的包含与等价
事件 $A$ 包含于事件 $B$，记作 $A \subseteq B$，如果 $A$ 发生必然导致 $B$ 发生。
事件 $A$ 与事件 $B$ 等价，记作 $A = B$，如果 $A$ 发生当且仅当 $B$ 发生。

### 和事件与积事件
和事件（并事件）是指两个事件中至少有一个发生的情况，记作 $A \cup B$。
积事件（交事件）是指两个事件同时发生的情况，记作 $A \cap B$。

### 差事件与对立事件
差事件是指事件 $A$ 发生而事件 $B$ 不发生的情况，记作 $A - B$。
对立事件是指事件 $A$ 不发生的情况，记作 $\bar{A}$。
所以 $A - B = A \cap \bar{B}$。

### 互斥事件和互逆事件
互斥事件是指两个事件不能同时发生的情况，记作 $A \cap B = \emptyset$。
互逆事件是指两个事件中一个发生时另一个必不发生的情况，记作 $A = \bar{B}$。

### 事件的运算律
事件的运算遵循以下基本律：
1. 交换律：
   - $A \cup B = B \cup A$
2. 结合律：
   - $(A \cup B) \cup C = A \cup (B \cup C)$
3. 分配律：
   - $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
